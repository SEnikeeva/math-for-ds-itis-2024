# math-for-ds-itis-2024
Математика и программирования для Data Science. Осень 2024.

Семестровый курс с 01.11.24 по 09.02.25
- Лекции по пятницам в 17:30
- Практики по субботам 15:50 - 19:00

## План на ~14 лекций 

1. Градиентные методы оптимизации
    - (01.11.24) Нейронные сети, функции активации, обратное распространение ошибки, SGD
    - (08.11.24) Модификации SGD
    - (15.11.24) Сходимость SGD
2. Матричные методы
    - (22.11.24) Малоранговая аппроксимация, SVD
    - (29.11.24) Применения SVD: PCA, RecSys, сжатие нейронок
    - (06.12.24) Линейная регрессия, LU-разложение, обратная матрица
    - (13.12.24) Число обусловленности, QR-разложение, МНК
    - (20.12.24) Разреженные матрицы
3. (27.12.24) Вероятностные распределения
4. (10.01.25) Параметрические оценки
5. (17.01.25) Марковские процессы
6. (31.01.25) Энтропия
7. (07.02.25) Преобразование Фурье / Консультация перед зачетом

**Баллы:**
- 10 домашних заданий по 10 баллов. Мягкий дедлайн - 2 недели, жесткий - 3 недели (половина баллов), после жесткого дедлайна сдавать нельзя.
- Доп. задания без дедлайнов.
- По 0.5 баллов за ответы на вопросы во время лекций (будут помечаться особым образом, на них нужно будет ответить сообщением в тг в беседу)
- Задачки со звездочкой из google classroom по 7 баллов 

**Зачет:**
- Допуск на зачет: задачки из google classroom (кроме задач со звездочкой), >60 баллов в сумме
- \>90 баллов автомат
- \>80 баллов полуавтомат
